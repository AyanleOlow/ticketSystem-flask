#  Ticket System – Flask

Dette prosjektet er et enkelt ticket-system laget med **Python** og **Flask**.  
Brukere kan sende inn tickets (forespørsler) gjennom en nettside.

Prosjektet er laget for å lære hvordan Flask fungerer og hvordan man bygger en enkel web-app.

---

##  Hvordan kjøre prosjektet

###  Det du trenger

- Python (3+)
- Git
- Terminal / CMD

---

###  Last ned prosjektet

```bash
git clone https://github.com/AyanleOlow/ticketSystem-flask.git
cd ticketSystem-flask
```
 Installer Flask
```bash
pip install flask
```
 Start serveren
```bash
flask run
```
 Åpne i nettleser
Gå til:
```bash
http://localhost:5000
```
